# Agrinho---2026
O objetivo central do projeto é educar e conscientizar sobre a transição ecológica no agronegócio, mostrando a evolução de uma fazenda tradicional (poluente) para uma fazenda do futuro (sustentável e tecnológica).

📝 Descrição do Projeto
O projeto funciona como um jogo ou infográfico animado em 2D. Ele utiliza elementos visuais mutáveis, cores e interatividade com o usuário para contar uma história em três capítulos (fases). A mensagem principal é mostrar como a tecnologia de precisão (como drones) e fontes de energia renovável podem transformar o impacto ambiental da produção de alimentos.
